# data-redacted — Sample Deliverables

This folder contains redacted samples of the full client deliverable suite produced for a Singapore construction sector credit risk engagement. All identifying information — company name, registration number, director names, address, bank name, counterparty details, and litigation case references — has been removed in compliance with a binding Non-Disclosure Agreement. Financial ratios, trend data, and analytical methodology are drawn from verified SCCB bureau reports and audited financial statements.

| File | Type | Description |
| :--- | :---: | :--- |
| `Construction Sector Credit Risk Analysis – Sample.pdf` | PDF | The primary client-facing report. Covers the full V-shaped recovery analysis: 3-year income statement, balance sheet, cash flow spreads, all profitability/liquidity/leverage/efficiency ratios, payment behaviour calendar, and the 5-step forensic dispute strategy. |
| `Construction_Sector_Credit_Report_Sample.docx` | DOCX | Full-length source document behind the PDF. Includes extended narrative sections, SCCB rating factor breakdown, litigation analysis, governance assessment, and the complete 11-point recommendation roadmap with scenario projections. |
| `Construction_Sector_Credit_Progress_Audit_Sample.docx` | DOCX | The 6-month follow-up audit document. Tracks implementation of all 11 recommendations against baseline metrics, applies the Green/Amber/Red verdict framework, and produces the next action plan for the October 2026 – March 2027 cycle. |
| `Corporate_Credit_Risk_Audit_Kit_Sample.xlsx` | XLSX | The working model handed to the client. Five-sheet kit: Instructions → Data Entry (FY2026 input cells) → Ratio Calculator (auto-updating) → Progress Tracker (before/after with scenario narratives) → Recommendation Checklist (dropdown status tracking). |

> All figures are real and sourced from verified bureau and financial statement data. The analytical methodology, ratio logic, and governance framework are proprietary to Henry C. Dibie and developed bespoke for this engagement.
